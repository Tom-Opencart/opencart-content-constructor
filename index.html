Кнопки в шапке конструктора

В верхней части экрана расположены кнопки управления. Вот что делает каждая из них:

Кнопка Что делает
Справка Открывает это руководство — вы сейчас здесь
Копировать код Копирует готовый HTML-код статьи в буфер обмена. Подходит, если вы хотите вставить код вручную в редактор OpenCart
Скачать TXT Скачивает готовый HTML-код в текстовом файле. Удобно сохранить для себя или передать кому-то
Скачать ZIP статьи Самая полезная кнопка! Собирает HTML-код + все картинки в один ZIP-архив. Этот архив можно загрузить на сайт
</> Export HTML Скачивает полноценный HTML-файл для просмотра в браузере. Удобно проверить, как статья будет выглядеть на сайте
{ } Export CSS Скачивает файл стилей. Нужен один раз для подключения стилей к сайту (см. инструкцию ниже)
Импорт JSON Загружает ранее сохраненный JSON-файл шаблона, полностью восстанавливая все блоки и настройки в конструкторе
Экспорт JSON Скачивает резервную копию вашей текущей статьи в формате JSON. Полезно для переноса на другой ПК или сохранения черновика
Шаблоны Выпадающий список с готовыми структурами страниц («Обзор товара», «Инструкция», «Услуги и FAQ») для быстрого старта
Превью Показывает или скрывает панель предварительного просмотра справа

Как загрузить созданную статью на сайт

Вы собрали статью в конструкторе и хотите показать её на своём сайте. Вот самая простая инструкция — следуйте пунктам по порядку:

Шаг 1. Подключите стили к сайту (один раз)

Чтобы ваша статья выглядела красиво (шрифты, отступы, таблицы, кнопки), нужно один раз подключить стили. Есть два способа — выберите любой:

Способ А: Автоматический (рекомендуется)

Скачайте ZIP-файл для установки
Нажмите кнопку — скачается ZIP-архив. Затем загрузите его на сайт (см. инструкцию ниже).
Что делать со скачанным ZIP-файлом:
  1. Откройте админ-панель вашего сайта. Обычно это адрес ваш-сайт.ru/admin. Введите логин и пароль.
  2. Слева в меню нажмите «Расширения». В подменю выберите «Установка расширений».
  3. В правом верхнем углу нажмите синюю кнопку «Загрузить».
  4. В открывшемся окне найдите скачанный ZIP-файл (он называется content_constructor.ocmod.zip) и выберите его.
  5. Дождитесь сообщения об успешной загрузке.
  6. Снова слева в меню нажмите «Расширения», но теперь выберите «Менеджер модификаторов».
  7. В правом верхнем углу нажмите синюю кнопку «Обновить» (с круговыми стрелками).
  8. Готово! Стили подключены.

Способ Б: Ручной

  1. Нажмите кнопку { } Export CSS в шапке конструктора — скачается файл content-constructor.css.
  2. Откройте этот файл любым текстовым редактором (Блокнот, Notepad++). Выделите весь текст (Ctrl+A) и скопируйте (Ctrl+C).
  3. Подключитесь к вашему сайту по FTP (или через файловый менеджер в админке).
  4. Найдите файл стилей вашей темы: catalog/view/theme/название_темы/stylesheet/stylesheet.css
  5. Откройте его и вставьте скопированные стили в самое конец файла.
  6. Сохраните файл.

Шаг 2. Сохраните статью и загрузите на сайт

Теперь, когда стили подключены, самое время загрузить вашу статью. Есть два варианта:

Вариант 1: Через кнопку импорта в админке (самый простой)

Этот вариант работает, если вы установили ZIP-файл (Способ А выше).

  1. В конструкторе нажмите «Скачать ZIP статьи». Скачается ZIP-архив с HTML-кодом и картинками.
  2. Откройте админ-панель сайта и перейдите к редактированию товара или статьи, куда хотите вставить контент.
  3. Найдите текстовый редактор (Summernote). В его панели инструментов нажмите группу «Вставить» (последняя иконка в ряду).
  4. Среди иконок найдите зелёную иконку файла (). Нажмите на неё.
  5. Выберите скачанный TXT-файл из конструктора (или ZIP — откроется файл content-{slug}.html).
  6. Контент автоматически загрузится в редактор!
  7. Нажмите «Сохранить».

Вариант 2: Вставка кода вручную

Если кнопка импорта не появилась или вы предпочитаете вставлять код сами:

  1. В конструкторе нажмите «Копировать код».
  2. Откройте редактор товара или статьи в админке OpenCart.
  3. Переключите редактор в режим HTML (обычно это кнопка «<>» или «Исходный код» на панели инструментов).
  4. Удалите всё, что есть в поле, и вставьте скопированный код (Ctrl+V).
  5. Переключите редактор обратно в обычный режим (кнопка «<>» ещё раз).
  6. Проверьте результат и нажмите «Сохранить».

Шаг 3. Проверьте результат

  1. Откройте страницу товара или статьи на вашем сайте (не в админке, а на самом сайте).
  2. Если контент отображается некрасиво — нажмите Ctrl + F5 для очистки кэша браузера.
  3. Если всё ещё некрасиво — убедитесь, что стили подключены (Шаг 1).

Частые ошибки:

  • Кнопка импорта не появилась → нажмите Ctrl + F5 на странице редактора
  • Картинки не отображаются → убедитесь, что домен в шапке конструктора совпадает с вашим сайтом
  • Стили не применяются → проверьте, что CSS-файл подключён (Шаг 1)

Что означают заголовки в шапке

Название статьи — вводится в первом поле в шапке. Используется как заголовок страницы при экспорте HTML и как основа для генерации слага.

Slug — автоматически генерируется из названия (например, apple-watch-obzor). Используется как имя файла при экспорте. Можно отредактировать вручную.

Домен — адрес вашего сайта (например, https://мой-магазин.ru/). Используется для отображения картинок в превью. Не влияет на сам контент.

Как изменить цвета

Если вам не подходят стандартные цвета конструктора, их можно быстро заменить в файле стилей content-constructor.css. Для этого откройте сохранённый CSS-файл в текстовом редакторе Notepad++ (или любом другом) и воспользуйтесь функцией массовой замены (сочетание клавиш Ctrl + H).

Основные цвета в коде:

  • #5446f8 — Основной синий цвет (цвет бордера слева от цитирования, активные вкладки, кнопки, ссылки).
  • #cfd8dc — Серый цвет рамки иконки вопроса в FAQ.
  • #78909c — Цвет самого знака вопроса ? внутри круглой иконки в FAQ.
  • #eaeaea — Цвет горизонтальных линий-разделителей под блоками FAQ.

Пример замены: Чтобы заменить основной синий цвет на свой (например, зеленый #27ae60), откройте файл CSS, нажмите Ctrl + H, введите в поле «Найти» значение #5446f8, а в поле «Заменить на» — #27ae60, после чего нажмите кнопку «Заменить все».

Как писать и оформлять текст

Конструктор понимает простые символы для форматирования. Вы пишете их обычным текстом, а конструктор автоматически превращает их в красивое оформление. Вот что можно делать:

Жирный текст

Оберните текст в две звёздочки с каждой стороны:

Вы пишете:
Это очень важная информация
На сайте будет:
Это очень важная информация

Когда нужно выделить ключевое слово, название продукта, цену или любую важную мысль.

Курсивный текст

Оберните текст в одну звёздочку с каждой стороны:

Вы пишете:
Модель Pro Max отличается увеличенным экраном
На сайте будет:
Модель Pro Max отличается увеличенным экраном

Для названий моделей, иностранных слов, цитат или терминов.

Ссылки

Напишите текст ссылки в квадратных скобках, а адрес — в круглых:

Вы пишете:
Подробнее на нашем сайте

или так:
[Подробнее на нашем сайте](https://example.ru/catalog)
На сайте будет:
Подробнее на нашем сайте

Ссылки на другие страницы вашего каталога, на инструкции, на сторонние ресурсы.

Маркированный список (с тире)

Каждый пункт начинайте с тире или минуса:

Вы пишете:
- Алюминиевый корпус
- Влагозащита IP68
- Батарея на 2 дня
На сайте будет:
  • Алюминиевый корпус
  • Влагозащита IP68
  • Батарея на 2 дня

Для перечисления характеристик, преимуществ, комплектации.

Нумерованный список (по порядку)

Каждый пункт начинайте с цифры и точки:

Вы пишете:
1. Включите устройство
2. Подключитесь к Wi-Fi
3. Следуйте инструкциям на экране
На сайте будет:
  1. Включите устройство
  2. Подключитесь к Wi-Fi
  3. Следуйте инструкциям на экране

Для пошаговых инструкций, порядка действий, этапов установки.

Список характеристик (Параметр :: Значение)

Введите два двоеточия между названием параметра и его значением:

Вы пишете:
Вес :: 52 грамма
Экран :: 1.9 дюйма AMOLED
Время работы :: до 18 часов
Водозащита :: 50 метров
На сайте будет:
Вес
52 грамма
Экран
1.9 дюйма AMOLED
Время работы
до 18 часов
Водозащита
50 метров

Идеально для описания характеристик товара — получается аккуратный список «название — значение».

Важно: Markdown-форматирование работает в блоках «Параграф», «Спойлер» и «Вкладки». В заголовках, цитатах и таблицах оно не нужно — там текст и так форматируется.

Сетка — размещение контента в колонки

Блок «Сетка» позволяет расположить картинку слева, а текст справа (или наоборот). Или разбить контент на 3-4 колонки. Это как таблица, но для произвольного контента.

Как использовать — пошагово:

  1. В левой панели нажмите «Сетка» — в рабочей области появится блок с пустыми колонками.
  2. Нажмите на блок сетки кнопку «Редактировать» (карандаш) и настройте:
    • Сколько колонок: 2, 3 или 4
    • Ширина колонок: например, 6 + 6 — две равные половины; 4 + 4 + 4 — три равных трети; 3 + 9 — узкая левая и широкая правая
    • На телефоне: 1 в ряд (колонки по очереди) или 2 в ряд
  3. Перетащите нужные блоки (картинку, текст, цитату) из левой панели прямо внутрь серой области нужной колонки.

Спойлер (аккордеон) — скрытый текст

Спойлер прячет длинный текст под заголовком. Пользователь видит только заголовок и нажимает на него, чтобы развернуть содержимое. Очень удобно для вопросов и ответов.

Где использовать:

  • Раздел «Часто задаваемые вопросы (FAQ)»
  • Длинные описания характеристик, которые не всем нужны
  • Скрытые инструкции или подробности
Как выглядит для пользователя:
Можно ли использовать эти часы с Android?
К сожалению, нет. Apple Watch работают только с iPhone. Для активации нужен iPhone модели 6s или новее.
Какое время работы от батареи?
До 18 часов в обычном режиме. При активном использовании GPS — до 7 часов.

Пользователь видит только заголовки. Нажал — развернулся текст.

Вкладки — группировка контента

Вкладки позволяют разнести информацию по нескольким «страничкам» внутри одного блока. Пользователь нажимает на вкладку и видит только её содержимое.

Где использовать:

  • «Характеристики» / «Описание» / «Комплектация» — для товара
  • «Для кого» / «Противопоказания» / «Дозировка» — для лекарств
  • Любая информация, которую удобно разделить на категории
Как выглядит для пользователя:

Экран 1.9", процессор S9, вес 52 г, водозащита IP68, аккумулятор 410 мАч.

Смарт-часы нового поколения с продвинутым датчиком пульса, встроенным GPS и уведомлениями от смартфона.

В комплекте: смарт-часы, магнитный кабель зарядки, инструкция по эксплуатации.

Пользователь переключается между вкладками одним кликом.

Совет: Во время редактирования вкладок можно добавлять и удалять их, менять заголовки и содержимое. Каждая вкладка — это отдельное поле для текста.

Инфо-блок (Callout / Скачивание файла)

Инфо-блок позволяет выделить важную информацию, ссылки или файлы для скачивания в красивую плашку с фоном, акцентной рамкой и кнопкой действия.

Доступно 4 стиля оформления:

  • Серый блок (Well): Универсальная классическая плашка для файлов.
  • Синий инфо-блок (Info): Для полезной информации и заметок.
  • Зеленый блок успеха (Success): Для скачивания документов, лицензий или успешных уведомлений.
  • Желтый предупреждающий блок (Warning): Для важных предупреждений, на которые пользователь должен обратить внимание.
Возможности настройки:
  • Выбор иконки (скачивание, ссылка, PDF, внимание, информация).
  • Настройка заголовка, текста описания и ссылки кнопки.
  • Адаптивная верстка (на мобильных устройствах кнопка и текст автоматически выстраиваются в один столбец для удобства).

Alert блок — цветные плашки примечаний

Этот блок позволяет быстро выделить предупреждение, совет или примечание ярким цветом, привлекая внимание пользователя. Полностью совместим со стандартными классами Bootstrap 3.

Где использовать:

  • Синий (Info): Важные заметки, ссылки на справочные материалы или сопутствующий контент.
  • Зеленый (Success): Примеры правильного выполнения действий, успешные результаты или бонусы.
  • Желтый (Warning): Предупреждения о возможных рисках или нюансах (например: "Перед началом работы отключите питание!").
  • Красный (Danger): Критические ошибки, запреты или ограничения, которые нельзя нарушать.

Адаптивное видео — YouTube и Vimeo

Вставка видеороликов с видеохостингов с автоматическим сохранением пропорций сторон 16:9 на любых экранах (от ПК до мобильных телефонов).

Как использовать:

  • Добавьте блок «Видео».
  • Вставьте полную ссылку на видео (например, https://www.youtube.com/watch?v=dQw4w9WgXcQ или короткую youtu.be).
  • Конструктор автоматически распознает платформу и извлечет ID видео для корректного отображения адаптивного фрейма.

Слайдер изображений — Карусель

Интерактивная галерея изображений с возможностью пролистывания слайдов, текстовыми подписями поверх картинок и точечными индикаторами.

Рекомендации по заполнению:

  • Старайтесь использовать картинки одинакового размера, чтобы избежать скачков высоты страницы при смене слайдов.
  • Заполняйте поле Alt текст для улучшения поисковой оптимизации (SEO) картинок.
  • Текстовые подписи к слайдам автоматически центрируются снизу на полупрозрачной подложке для лучшей читаемости.

Слайдер До / После — Сравнение картинок

Популярный интерактивный элемент, позволяющий пользователю плавно перетаскивать ползунок и сравнивать два изображения между собой (например, результат работы автосервиса, реставрации, тюнинга или ретуши).

Как сделать идеальный слайдер:

  1. Обе картинки ("До" и "После") должны иметь строго одинаковые пропорции и разрешение.
  2. Для корректного рендеринга элементы накладываются друг на друга в виде фоновых слоев, поэтому ползунок аккуратно срезает верхнее изображение.
  3. Заполняйте ярлыки (например, "До установки" / "После установки") для понятного визуального ориентирования.

Мессенджеры — Быстрая связь

Блок красивых и кликабельных кнопок с иконками для моментального перехода к переписке в WhatsApp, Telegram, Viber или совершения прямого звонка с мобильного телефона.

Формат ввода данных:

  • WhatsApp: Номер телефона в формате 79991234567 (без плюсов, пробелов и скобок).
  • Telegram: Имя пользователя без символа @ (например: username) или полная ссылка на аккаунт/канал.
  • Viber: Номер в формате 79991234567 (конструктор сам сформирует специальный URI viber://chat...).
  • Телефон: Номер в формате +79991234567 для осуществления вызова.

Карточка товара — Виджет для OpenCart

Специальный блок, предназначенный для внутренней рекламы товаров вашего магазина прямо в теле статьи. Помогает конвертировать читателей блога в покупателей.

Где и как использовать:

  • Вставляйте карточку товара в конце обзора или инструкции.
  • Указывайте точное название, актуальную цену и привлекательную картинку товара.
  • В поле ссылки укажите путь к товару (например: index.php?route=product/product&product_id=50 или ЧПУ-ссылку).
  • На мобильных устройствах карточка адаптивно перестраивается из горизонтального вида в вертикальный, чтобы изображение не сжималось.
  • Автозаполнение по ссылке: Вы можете вставить ссылку на страницу товара на вашем сайте и нажать «Автозаполнение». Конструктор автоматически импортирует название, цену и изображение.
    Важно: Для правильного импорта и превью картинок не забудьте указать адрес вашего магазина в поле «Домен» в шапке конструктора. Скрипт сам уберет из ссылки папки кэша (/image/cache/) и суффиксы размеров, оставив путь к оригиналу картинки (например, image/catalog/product.jpg), который корректно отобразится после загрузки статьи на ваш сайт.

Как добавлять картинки

Есть два способа работать с картинками. Выбирайте тот, который удобнее:

Способ 1: Указать путь к картинке на сервере

Если картинка уже загружена на ваш сайт (или вы знаете, где она лежит), укажите путь к ней:

Примеры путей:
image/catalog/my-product.jpg — картинка в папке каталога
image/catalog/photos/watch-front.png — картинка в подпапке
https://example.ru/picture.jpg — полная ссылка на интернет-ресурс

В превью картинка будет отображаться только если в шапке конструктора указан домен вашего сайта (например, https://ваш-сайт.ru/).

Способ 2: Загрузить картинку с компьютера

Самый простой способ — загрузить картинку прямо из конструктора:

  1. Добавьте блок «Изображение» в конструктор.
  2. Нажмите «Редактировать» (карандаш).
  3. В поле «Источник изображения» выберите «Загрузить локальный файл».
  4. Нажмите «Выберите файл» и найдите картинку на компьютере.
  5. Картинка появится в превью!

Что происходит «под капотом»: Конструктор запоминает картинку как специальный код (base64). При экспорте ZIP-архива картинка автоматически сохраняется в правильную папку на сервере. Вам не нужно ничего делать вручную!

Что делать с готовой картинкой:

  • «Скачать ZIP статьи» — лучший вариант. Конструктор соберёт все картинки и HTML-код в один ZIP-файл. После загрузки на сервер через «Установку расширений» всё будет работать.
  • «Копировать код» или «Скачать TXT» — скопирует HTML-код. Вам нужно будет вручную загрузить картинки на сервер через FTP или файловый менеджер OpenCart.

Кнопки-ссылки для открытых статей (agree)

В OpenCart есть функция показывать информационные страницы (например, «Политика конфиденциальности» или «Условия доставки») во всплывающем окне. Конструктор позволяет сделать такую ссылку на любое слово в тексте.

Как сделать ссылку на статью:

Простой формат:
[текст ссылки](agree:номер_статьи)

Пример 1:
[политикой конфиденциальности](agree:3)
На сайте появится ссылка: политикой конфиденциальности — при клике откроется статья с номером 3.

Пример 2:
[условиями доставки](agree:6)
На сайте появится ссылка: условиями доставки — при клике откроется статья с номером 6.

Пример 3:
[правилами возврата](agree:12)
На сайте появится ссылка: правилами возврата — при клике откроется статья с номером 12.

Где найти номер статьи (ID)?

  1. В админке OpenCart зайдите в «Каталог» → «Статьи (Информация)».
  2. Найдите нужную статью — например, «Политика конфиденциальности».
  3. Посмотрите на число в первой колонке — это и есть ID (номер). Запишите его.
  4. Используйте этот номер в ссылке: [ваш текст](agree:номер).

Альтернатива: отдельная кнопка-попап

Если хотите сделать не текстовую ссылку, а яркую кнопку — используйте блок «Попап окно» из левой панели:

  1. Добавьте блок «Попап окно».
  2. Введите текст кнопки (например, «Принять условия»).
  3. Укажите номер статьи (ID).
  4. Выберите стиль: текстовая ссылка или кнопка.
  5. Если выбрали кнопку — можно выбрать цвет (синий, зелёный, красный) и размер (маленькая, обычная, большая, на всю ширину).
Как выглядит кнопка:
Принять условия

При нажатии откроется всплывающее окно с текстом статьи из OpenCart.
