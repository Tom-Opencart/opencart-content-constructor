Интеграция с OpenCart 3

Автоматическая установка в OpenCart 3 (Рекомендуется)

Мы объединили все доработки в один удобный интеграционный ZIP-пакет. Он автоматически установит глобальный файл стилей на ваш сайт и добавит зелёную кнопку импорта в редактор Summernote.

Интеграционный ZIP-архив (.ocmod.zip)
Включает модификатор install.xml и файл стилей content-constructor.css в папке upload/.

Инструкция по установке ZIP-архива:

  1. Скачайте архив по кнопке выше. Он получит имя content_constructor.ocmod.zip.
  2. Перейдите в админке OpenCart в раздел: Расширения -> Установка расширений.
  3. Загрузите скачанный ZIP-архив.
  4. Перейдите в Менеджер модификаторов и нажмите кнопку «Обновить» в правом верхнем углу.
  5. После этого очистите кэш вашего браузера (кликните Ctrl + F5 на странице с редактором Summernote), чтобы обновить кэш редактора.

Ручные способы настройки

Если вы предпочитаете ручную установку или хотите использовать кастомные настройки, воспользуйтесь инструкциями ниже:

1. Как подключить стили статьи вручную

Правило: Стили не нужно загружать в каждую статью по отдельности! Подключите их один раз глобально:

  • Вариант А: Скопируйте CSS (кнопка Export CSS в шапке) и вставьте в настройки вашего шаблона (Custom CSS) в админке OpenCart.
  • Вариант Б: Допишите скопированный CSS в конец файла стилей темы (обычно catalog/view/theme/YOUR_THEME/stylesheet/stylesheet.css).
2. Отдельные XML-файлы модификаторов
Модификатор кнопки импорта в Summernote (admin/footer.twig)
Модификатор подключения стилей в head (catalog/header.twig)
3. Как повесить всплывающее окно (agree) на слово в тексте?

Для добавления всплывающего соглашения OpenCart к слову или фразе внутри текста выделите фрагмент в поле редактора, укажите ID статьи и нажмите кнопку agree. Также можно использовать стандартный синтаксис markdown-ссылок:

[Ваш текст](agree:ID)

Где вместо ID укажите номер информационной статьи из OpenCart. Например:

  • [политикой конфиденциальности](agree:3) — откроет в попапе статью с ID = 3.
  • [условиями соглашения](agree:5) — откроет в попапе статью с ID = 5.

При клике на такую ссылку в превью отобразится отладочное сообщение, а на живом сайте OpenCart она автоматически преобразуется в стандартное всплывающее окно.
