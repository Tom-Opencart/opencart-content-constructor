Руководство пользователя и интеграция

Автоматическая установка (Рекомендуется)

Для интеграции конструктора с вашим сайтом на OpenCart 3 скачайте готовый ZIP-архив. Он содержит все необходимые стили и добавляет удобную кнопку импорта в стандартный текстовый редактор Summernote в админке.

Интеграционный ZIP-архив (.ocmod.zip)
Содержит файл стилей content-constructor.css и модификатор install.xml.

Пошаговая инструкция для новичков:

  1. Нажмите зелёную кнопку выше и скачайте файл content_constructor.ocmod.zip.
  2. Войдите в админ-панель вашего сайта OpenCart.
  3. Перейдите в раздел «Расширения» ➔ «Установка расширений» и загрузите скачанный файл.
  4. Перейдите в раздел «Расширения» ➔ «Менеджер модификаторов» и нажмите кнопку «Обновить» (синяя кнопка с круговыми стрелками в верхнем правом углу).
  5. После этого очистите кэш вашего браузера (кликните Ctrl + F5 на странице с редактором Summernote), чтобы обновить кэш редактора.

Ручная настройка стилей

Если вы не хотите устанавливать плагин, вы можете подключить стили статьи вручную:

  • Нажмите кнопку Export CSS в шапке конструктора и скопируйте код.
  • Вставьте его в поле «Пользовательские CSS стили» в настройках вашей темы OpenCart или в конец файла стилей вашего шаблона (обычно catalog/view/theme/название_темы/stylesheet/stylesheet.css).

Списки параметров (Списки определений)

Идеально подходит для перечисления характеристик или терминов в формате «Параметр :: Значение». Вставьте двоеточия :: между параметром и его описанием:

Пример ввода:
Время работы :: До 18 часов в активном режиме
Дисплей :: Сенсорный OLED-экран Retina
Результат на сайте:
Время работы
До 18 часов в активном режиме
Дисплей
Сенсорный OLED-экран Retina

Сетка (Разметка на несколько колонок)

Позволяет разместить контент (например, картинку слева и текст справа) в несколько колонок.

  1. Добавьте блок «Сетка» в конструктор.
  2. В настройках блока выберите шаблон колонок (например, «6 + 6» для двух равных колонок, или «4 + 4 + 4» для трех).
  3. Перетаскивайте другие блоки (Параграфы, Картинки, Цитаты) из левой панели прямо внутрь появившихся пустых серых полей колонок.
  4. В экспортируемом коде используется сетка Bootstrap 3, которая автоматически адаптируется под экраны телефонов (колонки выстраиваются вертикально).

Спойлеры и Вкладки (Tabs)

Помогают скрыть длинные тексты, чтобы статья выглядела аккуратно:

  • Спойлер (Аккордеон): Скрывает текст под заголовком. Разворачивается при клике на стрелочку. Отлично подходит для раздела «Часто задаваемые вопросы (FAQ)».
  • Вкладки: Группируют контент по вкладкам. Пользователь может переключаться между ними в один клик (например: «Характеристики», «Описание», «Комплектация»).

Работа с изображениями

Поскольку изображения хранятся на вашем сервере OpenCart, следуйте этим рекомендациям:

  • Относительный путь: В поле пути к картинке вводите адрес относительно корня сайта, например: image/catalog/articles/my-photo.jpg.
  • Временный просмотр: Чтобы картинка отображалась в превью во время сборки, вы можете:
    1. Ввести в шапке конструктора домен вашего сайта (например, https://myshop.ru/), и он автоматически подставится в превью.
    2. Или нажать кнопку «Выбрать локальный файл» и загрузить картинку с компьютера (она отобразится в превью временно, но при экспорте останется правильный относительный путь).

Всплывающие статьи OpenCart (agree)

В OpenCart есть функция показа информационных статей (например, «Условия соглашения» или «Политика конфиденциальности») во всплывающем окне (попапе). Вы можете привязать такой попап к любому слову в тексте!

Для создания такой ссылки используйте формат:

[Ваш текст соглашения](agree:ID)

Где вместо ID укажите числовой номер информационной статьи из админки OpenCart (Статьи ➔ ID).

Примеры:

  • [политикой конфиденциальности](agree:3) ➔ откроет статью с ID = 3.
  • [условиями доставки](agree:6) ➔ откроет статью с ID = 6.
