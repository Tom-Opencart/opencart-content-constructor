# Руководство по подключению стилей статей в OpenCart 3

При переносе статей из конструктора в OpenCart крайне важно соблюдать правильный подход к работе со стилями (CSS).

> [!IMPORTANT]
> **Главное правило:** Стили статьи (CSS) **не нужно** вставлять внутрь каждой создаваемой страницы или статьи в базе данных. 
> Стили должны быть загружены на сайт **один раз глобально**.

Если вставлять стили (тег `<style>`) внутрь описания каждой статьи:
1. Это приведет к дублированию кода в базе данных и раздуванию размера страниц.
2. Если в будущем вы захотите изменить цвет ссылок или шрифт во всех статьях, вам придется вручную перенастраивать каждую статью в админке.

---

## Способ 1. Через настройки шаблона (Самый простой и рекомендуемый)

Большинство современных профессиональных шаблонов для OpenCart 3 (UniShop2, Journal 3, UltraStore, CyberStore и др.) имеют встроенное поле для пользовательских стилей.

1. Зайдите в панель управления вашим шаблоном в админке OpenCart.
2. Найдите вкладку **«Свой CSS»**, **«Кастомные стили»**, **«Custom CSS»** или **«Код»**.
3. Нажмите в конструкторе кнопку **«Export CSS»**, скопируйте полученный код и вставьте его в это поле.
4. Сохраните настройки шаблона. Теперь стили применятся ко всем статьям на сайте.

---

## Способ 2. Подключение внешнего CSS-файла через OCMOD (Профессиональный)

Если ваш шаблон не поддерживает кастомный CSS из админки, или вы хотите держать стили конструктора изолированно, можно подключить их как внешний файл стилей.

Мы подготовили для вас готовый файл модификатора: **[article_styles.ocmod.xml](file:///c:/Users/tomop/Downloads/extra%20news/article-constructor/article_styles.ocmod.xml)**.

### Инструкция:
1. Нажмите кнопку **«Export CSS»** в конструкторе и сохраните файл как `article-constructor.css`.
2. Загрузите этот файл на ваш сервер по пути:
   `catalog/view/theme/default/stylesheet/article-constructor.css`
   *(Если вы используете сторонний шаблон, вместо `default` может быть папка вашего шаблона, но модификатор по умолчанию ищет в `default` как в базовой папке)*.
3. Установите модификатор **[article_styles.ocmod.xml](file:///c:/Users/tomop/Downloads/extra%20news/article-constructor/article_styles.ocmod.xml)** через установщик дополнений OpenCart и обновите кэш модификаторов.
4. Модификатор автоматически добавит подключение этого файла во все шапки (`<head>`) вашего сайта.

---

## Способ 3. Добавление стилей в файл стилей темы (Классический)

Если вы не хотите использовать дополнительные модификаторы и настройки:

1. Откройте по FTP основной файл стилей вашего шаблона. Обычно он находится по пути:
   `catalog/view/theme/ТЕМА/stylesheet/stylesheet.css`
2. Перейдите в самый конец этого файла.
3. Добавьте скопированные из конструктора стили (нажав **«Export CSS»**) в конец файла.
4. Сохраните файл и обновите кэш браузера.
