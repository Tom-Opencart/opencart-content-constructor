# Конструктор контента для OpenCart 3 (Content Constructor)

Это удобный визуальный редактор (конструктор блоков), который позволяет быстро собирать статьи, обзоры товаров и посадочные страницы для OpenCart 3 без знания HTML и CSS. Сгенерированный код идеально подстроен под встраивание в стандартные описания товаров, категорий или статей.

Конструктор можно запускать локально или использовать готовую онлайн-версию:
👉 **<a href="https://tom-opencart.github.io/opencart-content-constructor/" target="_blank">Открыть конструктор на GitHub Pages</a>**

## 🚀 Быстрый старт: Как перенести дизайн на сайт (Простыми словами)

Если вы впервые пользуетесь конструктором, вот 3 простых шага, чтобы ваш дизайн появился на сайте:

1. **Один раз подключите стили к вашему сайту**:
   * В правом верхнем углу работающего конструктора нажмите кнопку **«Справка»**.
   * Нажмите кнопку **«Скачать ZIP-файл»** (скачается `content_constructor.ocmod.zip`).
   * Загрузите этот архив в админке OpenCart через меню **«Установка расширений»** и обновите модификаторы в **«Менеджере модификаторов»**. Теперь ваш сайт умеет красиво отображать сетки, табы и спойлеры (FAQ).

2. **Создайте и скачайте статью**:
   * Соберите нужные блоки и оформите текст в конструкторе.
   * Нажмите кнопку **«Скачать ZIP статьи»** в шапке конструктора (скачается ZIP-архив с кодом и всеми картинками).

3. **Загрузите её в OpenCart**:
   * Откройте редактирование товара или статьи в админке.
   * В текстовом редакторе (Summernote) нажмите появившуюся зелёную иконку импорта и выберите ваш скачанный файл (оранжевый TXT-файл статьи, либо HTML-файл из распакованного ZIP-архива статьи).
   * Сохраните изменения — всё готово!

## Основные возможности

1. **Визуальная сборка блоков**: заголовки, списки, таблицы, цитаты, картинки, табы, спойлеры (аккордеоны) и оглавление.
2. **Bootstrap 3 сетка**: множественный блок с секциями `col-xs-* col-md-*`, куда можно перетаскивать обычные блоки.
3. **Экспорт в один клик**:
   - **Копировать код** — копирует чистый HTML-код для вставки в текстовый редактор OpenCart.
   - **Export HTML** — скачивает полноценный HTML-файл для просмотра или архива.
   - **Export CSS** — скачивает файл стилей `content-constructor.css`.
   - **Экспорт в TXT** — сохраняет готовый HTML-код в файл для последующего импорта.
4. **Прямой импорт в OpenCart**: в редактор Summernote встраивается кнопка «Импорт», с помощью которой можно загрузить сохранённый файл контента одной кнопкой.
5. **Интеграция с всплывающими окнами (попапами) OpenCart**:
   - Можно создавать отдельные кнопки-попапы.
   - Можно привязывать попап соглашения (`agree`) прямо к любому слову в тексте.

## Как это работает?

### Шаг 1. Соберите контент
Добавьте нужные блоки на рабочую область редактора, заполните их текстом.
* Для форматирования текста (жирный, курсив, код) можно использовать простой Markdown.
* В полях ввода вы можете использовать списки или табы.

### Шаг 2. Как повесить всплывающее окно OpenCart (agree) на слово в тексте
Если вам нужно, чтобы при клике на слово в тексте открывалось модальное окно OpenCart (например, правила доставки или соглашение), выделите нужный фрагмент в поле редактора, укажите ID информационной статьи и нажмите кнопку **agree**.

Также можно написать это слово вручную в виде ссылки:
```markdown
[текст ссылки](agree:ID)
```
*Где `ID` — это числовой номер (ID) информационной статьи из вашей панели OpenCart.*
Пример: `[политикой конфиденциальности](agree:3)` или `[условиями доставки](agree:5)`.

### Шаг 3. Экспорт контента
Нажмите **Export HTML** и вставьте скопированный код в описание товара или статьи (в режиме просмотра кода в текстовом редакторе OpenCart). 
Для автоматического импорта файлов используйте кнопку импорта (см. интеграцию ниже).

## Простая интеграция с OpenCart 3 (Рекомендуется)

Мы подготовили автоматический установщик, который добавит кнопку импорта в редакторы Summernote в админке и подключит все необходимые стили.

1. Откройте Конструктор и нажмите кнопку **«Справка»** в правом верхнем углу.
2. Скачайте файл **`content_constructor.ocmod.zip`** по кнопке в модальном окне.
3. Перейдите в административную панель OpenCart в меню: **Расширения -> Установка расширений**.
4. Загрузите скачанный ZIP-архив.
5. Перейдите в **Менеджер модификаторов** и нажмите кнопку **«Обновить»** (синяя кнопка со стрелками вверху справа).
6. Очистите кэш браузера (`Ctrl + F5`) на странице с редактором Summernote.

После этого в текстовых редакторах Summernote с группой инструментов «Вставить» появится зелёная иконка файла. Нажмите на неё и выберите файл вашей статьи (это может быть оранжевый TXT-файл с кодом, скачанный по кнопке **«Скачать HTML-код в TXT-файле»** в шапке конструктора, либо HTML-файл с кодом из распакованного ZIP-архива статьи), после чего контент автоматически загрузится в редактор!
